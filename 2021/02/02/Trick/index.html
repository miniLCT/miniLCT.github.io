CF342 E. Xenia and Tree

this way

Description

给一棵树，一开始只有1为红，其他点为蓝。两种操作：

  1. 把一个点染成红点。
  2. 询问一个点到最近红点的距离。

Solution

数据范围$n\le 10^5,m\le 10^5$，我们不能每一次染色后都去跑一次最短路，所以我们当修改数目达到根号n时去进行一次最短路，查询是如果有点是未更新状态，我们可以通过 $lca$ 来求得两点的距离，这样就可以保证复杂度是$n\sqrt{n} \ log$。
